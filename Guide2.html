The Nav Bar

In order to add a Nav Bar you will need to use both HTML and CSS, firstly you will need to add the html. Which you can do by adding the opening and closing ul tags and then adding in it an opening and closing li tag then in that li tag you will add an a tag also known as a link tag.Remember that in order to make a tag you first need to type a left arrow. Then the name of the tag then an right arrow. Also in order to add the closing tag you first type a left arrow. Then you type a slash and then the name of the tag and the arrow. Before you close this link tag you will need to add the href attribute that we added earlier with the icon and exactly what website you want to go to with this link. Then close the opening tag and add what you want to the link to show up as in this case it could be home. Finally Close the a tag and li tag and close it in a ul tag. Now for the CSS in order to do this you will make a css document by calling the document (name).css and then linking that css file in the html with the link tag with the rel as stylesheet, type as text/css and href as the path to the file. Now it is linked to the html. In the css you will then type ul then a spikey bracket and a closing spikey bracket then in those brackets add list-style-type: none; margin: 0; and padding: 0; height: 20px; and background-color: blue; the list style type will remove the bullets and the margin 0 and padding 0 will remove the browsers default settings. Then height 20px will chang the height of the nav bar and backround-color blue will change the color of the nav bar. Then to make it horizontal you will will write li then the brackets and display: inline; this will display them in a line which will make it horizontal. After you do that you have to do some steps to make sure tha the text looks nice you need to add the display of li a to block then the width to 60px, text decoration to none and the color to red.
