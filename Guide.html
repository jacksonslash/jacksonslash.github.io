HTML Guide

The Software

The best way to make a website is to use either one of these softwares, brackets or sublime text 2 as these softwares provide a good base for html code. Then you will need to make a new document and call it (name).html with (name) as the name you want to give the file but remember it is best to call your homepage html document index.html as this is what most softwares use.

The Set Up

To start off you are going to need to add the structure for the website. This would mean adding the HTML Tags, Head Tags and Body Tags but what are these tags? A tag in html means a specific part of your website whether it be p tags for paragraphs or title tags for titles. The HTML Tag tells the computer that it is html while the head tags tell the computer that this is what is allowing the website to function and in what ways. Then the Body tags are the tags that are used for all the content that is on the website itself.

Adding a Title, Content and an Optional Icon

After you have made the structure you are going to want a title and some content in order to do this you will add 3 different tags, the title tag, the p tag and also the h1 to h6 tags. The title tag will make a title on the tab of your webpage. The titles whereabouts will be on the image below. The p tag like I said will make a paragraph and the h tags are a bit more complex. h1 is the biggest size you can get out of a header then it goes up to h6 which is the tiniest header. If you want you can also add an icon by doing the link tag then set the rel inside the tag to be equal to icon then set href equal to where the image is located.
